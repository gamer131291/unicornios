# 🦄 Unicornios Medievales - Página Web Premium

## 📜 Descripción
Página web interactiva que explora la fascinante historia de los unicornios durante la era medieval, basada en bestiarios, tapices y manuscritos históricos.

## ✨ Características
- **Diseño premium 2026** con glassmorfismo y efectos modernos
- **Totalmente responsive** (mobile, tablet, desktop)
- **Galería dinámica** de imágenes históricas
- **Modal de registro** listo para conectar con back-end
- **Animaciones suaves** y scroll reveal
- **Menú móvil** interactivo
- **Parallax effect** en hero section

## 🛠️ Tecnologías utilizadas
- HTML5 semántico
- CSS3 (Grid, Flexbox, Animaciones)
- JavaScript vanilla (ES6+)
- Font Awesome 6
- Google Fonts
